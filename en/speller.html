Speller

This is a spell checker made in C, optimized by choosing the proper data structures to work well with big workloads.
The program allows you to load your own word dictionaries to verify words from, as long as they're .txt files.
The picked data structure was a hash table, with each array containing a linked list, and each linked list consisting of a node.
Thanks to all these optimizations, we could reduce the average execution time from 10 seconds to less than 0.1 seconds.

// Hash function hashes a word into a number
 unsigned int hash(const char *w)
{
    char word[LENGTH + 1];
    for (int i = 0; i <= strlen(w); i++)
    {
        word[i] = tolower(w[i]);
    }
    unsigned int hash = 0;
    for (int i = 0, n = strlen(word); i < n; i++)
        hash = (hash << 2) ^ word[i];
    return hash % N;
}
